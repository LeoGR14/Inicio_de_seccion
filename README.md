# Inicio se Seccion

Proyecto frontend de una intranet escolar desarrollado con HTML, CSS y JavaScript.

## Descripción

Esta aplicación simula una intranet para un establecimiento educacional, permitiendo que estudiantes, profesores y administradores puedan acceder al sistema mediante un inicio de sesión y registro de usuarios.

## Funcionalidades

- Inicio de sesión mediante RUT y contraseña
- Registro de usuarios
- Recuperación de contraseña
- CAPTCHA de Google
- Formato automático de RUT chileno
- Validación de contraseña
- Confirmación de contraseña
- Mostrar / ocultar contraseña
- Detección de Bloq Mayús
- Validación visual de correo
- Bloqueo temporal por intentos fallidos
- Diseño responsive
- Animaciones visuales

## Tecnologías utilizadas

- HTML5
- CSS3
- JavaScript
- Google reCAPTCHA
